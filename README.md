### 🌉 Что означает VoirMost?
Voir (фр.) — "видеть", "смотреть"
Most (англ./рус.) — "мост"

---

## 🎥 Демонстрация экрана

Сердце проекта для программистов и творческих людей.

Ты показываешь свой код (или рисунок, чертёж) — другой участник смотрит, рисует поверх, предлагает решения.

**Два экрана:** твой и собеседника. Можно обсуждать в реальном времени.

> Идеально для парного программирования, ревью кода, совместных эскизов.
